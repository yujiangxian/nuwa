## 模块关系图

```
基础设施层 (1-4)
    ├─ 1.配置管理 ←→ 2.数据持久化 ←→ 3.文件存储
    └─ 4.服务生命周期 → 启动/关闭所有模块

AI引擎层 (5-8)
    ├─ 5.LLM推理 → 9.智能对话、10.Playground、15.工作流
    ├─ 6.TTS合成 → 9.智能对话、13.声音工坊
    ├─ 7.ASR识别 → 14.录音转写
    └─ 8.模型管理 → 5.LLM、6.TTS、7.ASR

业务功能层 (9-15)
    ├─ 9.智能对话 ← 5.LLM + 6.TTS + 7.ASR + 11.角色 + 12.预设
    ├─ 10.Playground ← 5.LLM + 12.预设
    ├─ 11.角色管理 ← 6.TTS(音色绑定)
    ├─ 12.提示词管理 → 9.智能对话、10.Playground
    ├─ 13.声音工坊 ← 6.TTS
    ├─ 14.录音转写 ← 7.ASR
    └─ 15.工作流编排 ← 5.LLM(Agent调度)

系统框架层 (16-19)
    ├─ 16.导航路由 → 所有业务模块的页面切换
    ├─ 17.主题国际化 → 全局样式
    ├─ 18.键盘快捷键 → 全局交互
    └─ 19.错误处理 → 全局异常捕获
```
