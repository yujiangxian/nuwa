| `material-symbols-api.css` | 2.4 KB | Google Fonts API 返回的完整 CSS（含 7 个字体子集 URL） |

**使用方式**：
```css
/* 在项目的全局 CSS 中引入 */
@import url('./material-symbols-api.css');

/* 然后使用 class */
<span class="material-symbols-outlined">settings</span>
<span class="material-symbols-outlined">mic</span>
<span class="material-symbols-outlined">play_arrow</span>
```

Material Symbols 支持 **4 个可变轴**：
- `opsz`（光学尺寸）: 20~48
- `wght`（字重）: 100~700
- `FILL`（填充）: 0~1
- `GRAD`（渐变）: -50~200

```css
.material-symbols-outlined {
  font-variation-settings: 'opsz' 24, 'wght' 400, 'FILL' 0, 'GRAD' 0;
}
/* 激活状态：填充 + 加粗 */
.active {
  font-variation-settings: 'opsz' 24, 'wght' 600, 'FILL' 1, 'GRAD' 0;
}
```

### 3.2 shadcn/ui 与 Material 3 的关系

shadcn/ui **不是** Material Design 组件库，它是基于 Tailwind + Radix UI 的无样式组件集合。但可以通过修改 CSS 变量和类名来**适配 Material 3 风格**：

```css
/* 将 shadcn/ui 按钮改为 Material 3 Filled 风格 */
.btn-filled {
  @apply bg-primary text-on-primary px-6 py-2.5 rounded-lg;
  @apply hover:shadow-md transition-shadow duration-150;
}
```

**推荐策略**：
- 用 **shadcn/ui** 做基础组件（已有 Tailwind + React 项目）
- 用 **Material 3 规范** 控制视觉风格（颜色、圆角、阴影、动效）
- 用 **Material Symbols** 做图标

## 七、Material 3 Expressive 核心原则（2025）

Google 2025 年最新设计方向：

- **Containment（容器）**：对比色区分交互元素，即时传达视觉结构
- **Common Region（共同区域）**：圆角容器明确分组
- **Stronger Colors（强色彩）**：按钮更醒目
- **Motion（动效）**：过渡动画传达层级
- **Shape & Size（形状与尺寸）**：大胆变化创造愉悦体验

### 2. 组件开发策略

**本项目推荐**（React + Tailwind）：
1. **基础组件**：从 shadcn/ui 复制源码到 `app/web/src/components/ui/`
2. **风格定制**：按照 Material 3 规范修改 Tailwind 类名（颜色、圆角、阴影）
3. **图标**：使用 Material Symbols（已下载 WOFF2）
4. **布局**：参考 Apple HIG 的侧边栏 + Material 3 的卡片容器

**具体步骤**：
```bash
# 1. 安装 shadcn/ui CLI（需网络）
npx shadcn@latest init

# 2. 添加所需组件
npx shadcn add button card dialog sheet sidebar tabs

# 3. 修改 globals.css，注入 Material 3 CSS 变量
# 4. 调整组件源码中的 Tailwind 类名，匹配 Material 3 风格
```

**离线替代方案**（无网络时）：
- 打开 `shadcn-Button.html`，复制其中的 TypeScript 源码
- 粘贴到 `app/web/src/components/ui/button.tsx`
- 修改 className 中的 Tailwind 类名
