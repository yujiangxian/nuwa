# 功能规格：声音工坊 (VoiceStudio)

---

## 1. 功能概述

一站式语音处理工作台，三个核心 Tab：
- **声音库**：管理已有音色，试听、切换默认音色
- **语音合成**：输入文本 → 调节参数 → 合成语音 → 播放下载
- **声音克隆**：上传参考音频 → 提取音色 → 保存为新声音

## 3. 录音转写 → 合成工作流

```
用户点击"开始录音"
    │
    ▼
MediaRecorder 录制 (audio/webm)
    │
    ▼
用户点击"停止录音"
    │
    ▼
Blob 上传 POST /api/inference/asr/upload
    │
    ▼
识别文本自动填入 textarea
    │
    ▼
用户编辑文本、调节参数
    │
    ▼
点击"开始合成"
    │
    ▼
POST /api/inference/tts
    │
    ▼
显示音频播放器，可播放/下载
```

---

## 4. 参数说明

| 参数 | 范围 | 默认值 | 说明 |
|------|------|--------|------|
| 语速 | 0.5x ~ 2.0x | 1.0x | 当前仅前端展示，后端 CosyVoice 暂不支持调节 |
| 音调 | -12 ~ +12 | 0 | 同上 |
| 情感 | 6 种 | 中性 | 同上，未来接入情感控制 TTS |

> ⚠️ 当前 TTS 后端（CosyVoice zero-shot）不支持语速/音调/情感参数调节。这些滑块是 UI 占位，接入 GLM-TTS 或 Qwen3-TTS 后可实现。

## 5. 声音克隆（未来）

```
用户上传 3-10 秒参考音频
    │
    ▼
前端上传至服务器
    │
    ▼
保存到 assets/voices/{name}/
    │
    ▼
可选：用 ASR 自动识别参考文本
    │
    ▼
保存为"新音色"，加入声音库
```

---

## 6. 已知问题

| 问题 | 方案 |
|------|------|
| 语速/音调/情感滑块无实际效果 | 接入支持这些参数的 TTS 模型（GLM-TTS / Qwen3-TTS）|
| 无音色库持久化 | 增加 voices 目录 + 后端 voices API |
| 合成后音频无法下载 | 增加下载按钮（直接链接 /api/audio/xxx.wav）|
